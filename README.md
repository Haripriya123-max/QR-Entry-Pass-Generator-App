Project Overview

The QR Entry Pass Generator is designed to manage entry access efficiently using QR codes.

It allows:

👩‍🎓 Students/Users to register and generate QR passes
🛡️ Security to scan and verify QR codes
🗂️ Admin/Security to view scan history and access records
This system improves security, reduces manual verification, and maintains digital entry logs.

🚀 Features

🔐 User Registration & Login
🎫 QR Code Generation
📷 QR Code Scanning & Verification
🕒 Entry Log / Scan History
👮 Role-Based Access (Student / Security)
🌐 Responsive UI
⚡ Real-time QR validation
